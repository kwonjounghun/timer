# Timer App

Vite 기반 React 타이머 애플리케이션입니다.

## 기능

- 시작/일시정지 버튼으로 타이머 제어
- 리셋 버튼으로 타이머 초기화
- 시:분:초 형식으로 시간 표시
- 반응형 디자인

## 설치 및 실행

1. 의존성 설치:
```bash
npm install
```

2. 개발 서버 실행:
```bash
npm run dev
```

3. 빌드:
```bash
npm run build
```

4. 미리보기:
```bash
npm run preview
```

## 기술 스택

- React 18
- Vite
- CSS3 (Glassmorphism 디자인)
- ESLint
